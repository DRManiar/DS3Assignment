This file describes the variables in the tidy data file named "DS3Assign_tidytable.txt". The data in this file are average values of mean and standard deviation of measurements grouped by each subject and each activity. The data size is 180 rows and 81 columns.

Units:
- The triaxial accelerations values and their magnitudes are in standard gravity units 'g'.
- The triaxial angular velocities and their magnitudes (the Gyro values) are in (rad/sec) or (1/sec).
- The acceleration jerk values are in ('g'/sec).
- The 'gyro' jerk values are in (rad/sec^2) or (1/sec^2).
- The meanFreq() variables are in Hz.

The column variable names are described below in the column order. Columns 3 to 81 contains numeric values and they all are avearge values as mentioned above.

activity -- Contain six unique activity labels (character)
